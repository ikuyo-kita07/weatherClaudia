# WeatherClaudia

Yet another weather app that provides current weather information using Open-Meteo API.

## Features

- Temperature unit management (Celsius/Fahrenheit)
- Coordinate management – manually enter latitude and longitude instead of granting location permission
- Current weather conditions powered by the Open-Meteo API
- Clean and minimal Material Design interface
